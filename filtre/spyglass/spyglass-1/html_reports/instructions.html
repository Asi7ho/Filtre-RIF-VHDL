Instructions - SpyGlass DashBoard on Google Chrome(Windows Platform)

  1. Locate your Google Chrome shortcut (On your desktop or in Start Menu - Any one will work)
  2. Right-Click this shortcut and choose Properties
  3. Find the target box (it will be highlighted when the properties screen appears)
  4. Left click at the end of the line (after chrome.exe) & Press space
  5. Add a command line switch(--allow-file-access-from-files).
    Append this Switch in Target Box (will read like this): C:\Users\USERNAME\AppData\Local\Google\Chrome\Application\chrome.exe --allow-file-access-from-files
  6. Press Apply/OK, restart Chrome and run the DashBoard
